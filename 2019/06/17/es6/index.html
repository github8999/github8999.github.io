1 块级作用域

ES6(ES2015) 中提出了块级作用域的概念

  • let 的使用规则

  1. let 声明的变量不会被变量提升

  2. let 不可以在同一个作用域中重复声明变量

  3. let 声明的变量会区分块状作用域( {}大括号包裹起来的代码块都表示为块状作用域 );

  • const 的使用规则

  1. const 一般用来声明常量, 使用 const 定义的常量不能修改(对象则是不能修改引用)

  2. const 定义的变量必须被初始化
